# Changelog

## 0.8.0-alpha — 2026-07-17

- Добавлен быстрый турнирный режим: запуск помощи за три нажатия.
- Главный экран перестроен вокруг действия «Помочь за 20 секунд».
- После выбора состояния и помехи голосовая практика начинается автоматически.
- Быстрый протокол сокращён примерно до 48 секунд.
- Задача на начало подбирается автоматически по ситуации.
- При совпадении ситуации используется успешная задача из прошлой схватки.
- Добавлена кнопка повтора последней удачной настройки.
- Полный сценарий сохранён под кнопкой «Сейчас всё по-другому».
- Добавлены крупные турнирные карточки с увеличенной областью нажатия.
- Добавлен визуальный индикатор трёх нажатий.

## 0.7.0-alpha — 2026-07-17

- Реализован полный цикл «до схватки → после схватки → следующая настройка».
- Каждая завершённая практика создаёт локальную карточку будущей схватки.
- Добавлена короткая структурированная обратная связь после схватки.
- Следующий протокол использует вывод из последней завершённой схватки.
- Добавлена история карточек и отдельный экран деталей.
- Добавлен профиль четырёх тренируемых навыков без единого психологического рейтинга.
- Разделены «Помощь сейчас» и «Тренировать заранее».
- Добавлен выбор одной задачи на первые десять секунд и персональной короткой команды.
- Расширена библиотека навыков: reset, ожидание, одна команда среди шума, сильный соперник.
- Данные переведены на отдельные модели и репозиторий локального хранения.
- Исходный код разделён на модули для дальнейшего развития.

## 0.6.0-alpha — 2026-07-17

- Рабочее название изменено на «Своя борьба».
- Добавлены четыре режима времени до схватки.
- Добавлены понятный чек-ин состояния, русская озвучка и локальная персонализация.
