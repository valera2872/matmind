# Changelog

## 0.4.0-alpha — 2026-07-17

- Подключены все ранее декоративные карточки родительского раздела.
- Добавлены полноценные сценарии перед стартом, при отказе ехать и для саморегуляции родителя.
- Карточка выбора теперь требует обработчик нажатия на уровне типа, поэтому новую «неработающую кнопку» нельзя случайно добавить.
- Возврат из каждого родительского сценария ведёт к списку ситуаций.

## 0.3.0-alpha — 2026-07-17

- Добавлен адаптивный чек-ин по контексту, энергии, напряжению, фокусу и ощущению контроля.
- Реализован объяснимый подбор протокола: reset после ошибки, снижение лишнего напряжения, активация, сужение фокуса или репетиция первого эпизода.
- После практики приложение измеряет изменение состояния и предлагает задачу переноса на ковёр.
- Результат адаптивной практики добавляется в дневник текущей сессии.
- В профиль добавлена статистика адаптивных практик.

## 0.2.0-alpha — 2026-07-17

- Все пять ситуаций главного экрана стали интерактивными.
- Добавлены сценарии «Сильный соперник», «Reset после ошибки», «После поражения» и «Не хочу ехать».
- Добавлена библиотека из пяти ежедневных психологических тренировок.
- Добавлен дневник спортсмена с созданием и удалением записей в текущей сессии.
- Добавлен профиль с возрастом, видами спорта и прогрессом.
- Нижняя навигация теперь работает.

## 0.1.0-alpha — 2026-07-17

- Первый Flutter-каркас MATMIND.
- Онбординг спортсмена и родителя.
- Возрастные версии 10–12, 13–15 и 16–17 лет.
- Главный сценарий «Я выхожу через пять минут».
- Демо-практика «Первые 30 секунд».
- Выбор одной процессуальной цели.
- Родительская карточка после поражения.
- GitHub Actions для сборки Android APK.
