# Своя борьба 0.8.0-alpha — быстрый старт

Flutter-прототип персональной психологической подготовки юных спортсменов борцовских видов спорта.

## Главная цель версии

**Полезная голосовая настройка за три нажатия.**

1. Нажать «Помочь за 20 секунд».
2. Выбрать состояние тела.
3. Выбрать главную помеху — практика запускается автоматически.

«20 секунд» означает путь до запуска помощи, а не продолжительность самой практики. Быстрая практика рассчитана примерно на 48 секунд.

## Что добавлено в 0.8.0

- турнирный главный экран с крупными зонами нажатия;
- быстрый режим без промежуточного экрана плана;
- автоматический выбор задачи на первые десять секунд;
- использование успешной прошлой задачи при совпадении ситуации;
- компактный голосовой протокол примерно на 48 секунд;
- кнопка «Повторить удачную настройку»;
- кнопка «Сейчас всё по-другому» для полного сценария;
- крупные карточки, пригодные для шумного зала;
- понятный индикатор трёх нажатий;
- сохранение полного цикла 0.7.0: карточки схваток, обратная связь, история и профиль навыков.

## Сборка APK через GitHub

1. Распакуйте архив.
2. Загрузите содержимое папки `svoya_borba_0.8.0` в корень репозитория.
3. Обязательно загрузите скрытую папку `.github`.
4. Откройте **Actions**.
5. Запустите **Build Android APK**.
6. Скачайте artifact `svoya-borba-0.8.0-debug-apk`.

Workflow создаёт Android-платформу, устанавливает зависимости, выполняет анализ, тесты и debug-сборку.

## Ограничения прототипа

- используется системный TTS;
- профессиональные аудиозаписи запланированы на 0.9.0;
- облачный AI не подключён;
- данные хранятся локально;
- приложение не заменяет тренера, врача или спортивного психолога.
