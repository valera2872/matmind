# MATMIND 0.3.0-alpha

Первая Flutter-alpha психологического тренажёра для юных борцов 10–17 лет и их родителей.

## Что реализовано

- выбор роли спортсмена или родителя;
- возрастные версии 10–12, 13–15 и 16–17 лет;
- выбор борцовского вида спорта;
- главный экран «Что происходит сейчас?»;
- рабочий путь «Я выхожу через пять минут»;
- чек-ин уровня возбуждения;
- демо-практика «Первые 30 секунд»;
- выбор одной процессуальной цели;
- родительская карточка после поражения.
- сценарий «Сильный соперник»;
- reset после ошибки;
- восстановление после поражения;
- безопасный разбор отказа ехать на соревнование;
- пять ежедневных психологических тренировок;
- дневник спортсмена в текущей сессии;
- рабочая нижняя навигация и профиль.
- адаптивный чек-ин по моменту, энергии, напряжению и фокусу;
- объяснимый подбор одного из пяти протоколов вместо общей мотивационной карточки;
- измерение управления состоянием до и после практики;
- конкретная задача переноса психологического навыка на ковёр;
- автоматическая запись результата практики в дневник текущей сессии.

## Принцип продукта

MATMIND строится как замкнутый тренировочный цикл: **оценить состояние → подобрать протокол → выполнить → применить на ковре → проверить эффект**. Победа и поражение не превращаются в «оценку психики» спортсмена.

Подробная продуктовая стратегия и сравнение с аналогами: [`docs/PREMIUM_PRODUCT_BLUEPRINT_RU.md`](docs/PREMIUM_PRODUCT_BLUEPRINT_RU.md).

## Как получить APK через GitHub

1. Создайте пустой репозиторий GitHub.
2. Загрузите в него всё содержимое этой папки, включая `.github`.
3. Откройте вкладку **Actions**.
4. Запустите workflow **Build Android APK** или дождитесь автоматической сборки.
5. Откройте завершившийся запуск и скачайте artifact `matmind-debug-apk`.

Workflow сам создаёт отсутствующую Android-платформу командой `flutter create`, затем проверяет и собирает приложение.
