# MATMIND

Первая Flutter-alpha психологического тренажёра для юных борцов 10–17 лет и их родителей.

## Что реализовано

- выбор роли спортсмена или родителя;
- возрастные версии 10–12, 13–15 и 16–17 лет;
- выбор борцовского вида спорта;
- главный экран «Что происходит сейчас?»;
- рабочий путь «Я выхожу через пять минут»;
- чек-ин уровня возбуждения;
- демо-практика «Первые 30 секунд»;
- выбор одной процессуальной цели;
- родительская карточка после поражения.

## Как получить APK через GitHub

1. Создайте пустой репозиторий GitHub.
2. Загрузите в него всё содержимое этой папки, включая `.github`.
3. Откройте вкладку **Actions**.
4. Запустите workflow **Build Android APK** или дождитесь автоматической сборки.
5. Откройте завершившийся запуск и скачайте artifact `matmind-debug-apk`.

Workflow сам создаёт отсутствующую Android-платформу командой `flutter create`, затем проверяет и собирает приложение.
